
# A simple Notes App 
## Group 30 

A simple Notes App for creating and editing sticky Notes made with Android Studio & Java


## Features
Uses Realm Db's client side to store the notes on the Phone's local storage


## Tech Stack

**IDE**: Android Studio

**Tools:**  Java, Realm DB
